# Database Folder
## محتويات هذا المجلد

هذا المجلد يحتوي على كل الملفات الخاصة بقاعدة البيانات للمشروع.

---

## 📁 الملفات الموجودة:

### 1. `schema.md`
**Database Schema Document** - وثيقة شاملة تحتوي على:
- تصميم جداول قاعدة البيانات (13 جدول)
- العلاقات بين الجداول
- الـ Indexes للأداء الأمثل
- 40+ API Endpoint
- استراتيجية Real-time Sync
- ملاحظات الأمان والـ Migration

**الاستخدام:** قدم هذا الملف للمبرمج المسؤول عن تطوير الـ Backend

---

## 📝 ملفات إضافية (ستضاف لاحقاً):

### 2. `migrations/` (مجلد)
سيحتوي على ملفات الـ SQL Migration لإنشاء وتحديث قاعدة البيانات

### 3. `seeds/` (مجلد)
سيحتوي على بيانات تجريبية (Mock Data) لملء قاعدة البيانات للتطوير والاختبار

### 4. `api-documentation.md`
توثيق تفصيلي لكل API Endpoint مع أمثلة على الـ Request/Response

### 5. `erd-diagram.png`
رسم بياني للعلاقات بين الجداول (Entity Relationship Diagram)
