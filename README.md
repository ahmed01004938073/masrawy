# مشروع أفليت للتجارة الإلكترونية

## معلومات المشروع

**تاريخ آخر تحديث**: 2023-11-15

## التحديثات الأخيرة

1. **تحديث قسم التسليم**:
   - إصلاح مشكلة إضافة العمولة للمسوق عند تسليم الطلب
   - إضافة عدد الطلبات للمسوق عند التسليم الكامل والجزئي
   - عدم إضافة العمولة عند رفض الاستلام

2. **تحسينات واجهة المستخدم**:
   - إزالة زر الإغلاق المكرر من صفحة تفاصيل الطلب
   - تحسين رسائل الخطأ والنجاح

3. **تنظيف الكود**:
   - إزالة التعليقات والكود غير المستخدم
   - تحسين أداء التطبيق

## كيفية تشغيل المشروع

### المتطلبات الأساسية

- Node.js (الإصدار 18 أو أحدث)
- npm (الإصدار 9 أو أحدث)

### خطوات التشغيل

1. قم بتثبيت الاعتماديات:
```sh
npm install
```

2. قم بتشغيل خادم التطوير:
```sh
npm run dev
```

3. افتح المتصفح على العنوان:
```
http://localhost:9000
```

## هيكل المشروع

```
afleet/
├── src/
│   ├── components/     # مكونات واجهة المستخدم القابلة لإعادة الاستخدام
│   ├── contexts/       # سياقات React للحالة العامة
│   ├── data/           # بيانات ثابتة ونماذج
│   ├── hooks/          # خطافات React المخصصة
│   ├── pages/          # صفحات التطبيق
│   ├── services/       # خدمات التخزين المحلي وواجهات API
│   ├── styles/         # أنماط CSS العامة
│   ├── App.tsx         # مكون التطبيق الرئيسي
│   └── main.tsx        # نقطة الدخول للتطبيق
├── public/             # الملفات الثابتة
└── package.json        # تكوين المشروع والاعتماديات
```

## الأقسام الرئيسية

1. **قسم الطلبات**: إدارة الطلبات الجديدة وتأكيدها
2. **قسم المخزن**: تجهيز الطلبات المؤكدة
3. **قسم الشحن**: إدارة عمليات شحن الطلبات
4. **قسم التوصيل**: متابعة حالة توصيل الطلبات
5. **قسم المسوقين**: إدارة المسوقين وعمولاتهم
6. **قسم المنتجات**: إدارة المنتجات وتفاصيلها

## التقنيات المستخدمة

تم بناء هذا المشروع باستخدام:

- Vite - إطار عمل لتطوير تطبيقات الويب
- TypeScript - لغة برمجة آمنة النوع
- React - مكتبة JavaScript لبناء واجهات المستخدم
- shadcn-ui - مكتبة مكونات واجهة المستخدم
- Tailwind CSS - إطار عمل CSS للتصميم السريع
- React Query - لإدارة حالة البيانات وطلبات API
- React Router - للتنقل بين الصفحات

## ميزات المشروع

1. **إدارة الطلبات**:
   - إنشاء وتعديل وحذف الطلبات
   - تتبع حالة الطلب من التأكيد إلى التسليم
   - عرض تفاصيل الطلب والمنتجات

2. **نظام المسوقين**:
   - إدارة المسوقين وعمولاتهم
   - حساب العمولات تلقائيًا عند تسليم الطلبات
   - تتبع أداء المسوقين وإحصائياتهم

3. **إدارة المخزون**:
   - تتبع المنتجات وكمياتها
   - تحديث المخزون تلقائيًا عند تأكيد الطلبات

4. **واجهة مستخدم سهلة الاستخدام**:
   - تصميم متجاوب يعمل على جميع الأجهزة
   - واجهة مستخدم بسيطة وسهلة الاستخدام
   - دعم اللغة العربية بالكامل
